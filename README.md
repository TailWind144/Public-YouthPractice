# YouthPractice

该项目是基于 `Nuxt3` 的全栈项目。本项目聚焦国内初高中英语学习场景，以 “游戏化闯关学习” 为核心设计理念，针对青少年学习动力不足、过程枯燥等痛点，提供了一个趣味与实效兼备的英语学习平台。项目的核心功能涵盖签到打卡、积分奖励、等级成长、每日一句、闯关练习等。

## 部署

下载依赖：

```
pnpm install
```

启动开发环境服务：

```
pnpm dev
```

构建：

```
pnpm build
```
